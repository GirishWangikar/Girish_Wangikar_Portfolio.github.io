Neural Network Intermediate Layer Visualization

This project aims to explore the visualization of intermediate activations in a Convolutional Neural Network, particularly using the classic LeNet-5 model on the MNIST dataset.

By understanding the learned patterns and representations, we hope to gain valuable insights into the inner workings of CNNs and their remarkable performance in image classification tasks. <

The Deconvnet approach and activation visualization provide a powerful toolset for model interpretability and Explainable AI, making this project an essential step towards understanding complex neural networks.
