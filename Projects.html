Deepfake Face Detection

The goal of this project is to explore and develop a model that can effectively detect deep fake face images generated from various GAN architectures.

The existing detection models often struggle to generalize across different GAN architectures.

This project aims to reimplement the pairwise learning model proposed by Hsu C. et al., which utilizes statistics between real and fake image pairs to improve detection performance.

Neural Network Intermediate Layer Visualization

Convolutional Neural Networks (CNNs) have proven to be highly effective in image classification tasks, but the inner workings of these networks and the specific patterns learned by each layer remain somewhat mysterious.

In this project, I aim to explore the visualization of the outputs of each convolutional layer to gain insights into the learned patterns and shed light on the remarkable performance of CNNs. Specifically, we will use the popular MNIST dataset and the classic 'LeNet5' model to classify handwritten digits.

IMDB Reviews Binary Classifier

This project aims to build a binary classifier that can predict whether a movie review is positive or negative based on the IMDB Reviews dataset.

The dataset consists of 100,000 movie reviews, divided into 'train', 'test', and 'unsupervised' sets.

For this project, only the 'train' and 'test' sets will be used, as they contain labeled examples necessary for training the model.
