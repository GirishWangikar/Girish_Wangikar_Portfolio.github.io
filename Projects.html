GraphRAG Q&A and Visualization System

I developed an interactive application that combines advanced language models with graph processing techniques to analyze and visualize relationships within text data. Deepfake Face Detection

The goal of this project is to explore and develop a model that can effectively detect deep fake face images generated from various GAN architectures.

The existing detection models often struggle to generalize across different GAN architectures.

This project aims to reimplement the pairwise learning model proposed by Hsu C. et al., which utilizes statistics between real and fake image pairs to improve detection performance.
